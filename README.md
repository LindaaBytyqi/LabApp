# 📚 Online Library System

**Online Library System** është një platformë web për menaxhimin dhe porositjen e librave online. Sistemi mbështet role të ndryshme të përdoruesve për menaxhim të plotë të bibliotekës digjitale.

---

## 🚀 Përshkrimi i projektit

Ky sistem mundëson:
- shfletimin dhe porositjen e librave nga përdoruesit
- menaxhimin e librave dhe kategorive nga administratori
- menaxhimin e porosive nga koordinatori

---

## 👥 Rolet në sistem

### 👤 User
- Regjistrim dhe login
- Shfletim i librave
- Kërkim sipas kategorive / autorëve
- Porositje e librave
- Shikim i historikut të porosive

---

### 🛠 Administrator
- Shton libra të rinj në sistem
- Menaxhon kategoritë e librave
- Menaxhon autorët
- Editon dhe fshin libra
- Menaxhon të dhënat kryesore të sistemit

---

### 📦 Koordinator
- Menaxhon porositë e librave
- Kontrollon statusin e porosive
- Përpunon dhe organizon kërkesat e përdoruesve
- Ndihmon në procesin e shpërndarjes së librave

---

## 🧰 Teknologjitë e përdorura

- ASP.NET Core Web API
- React.js
- TypeScript
- REST API
- SQL Database
- HTML, CSS
